Definição Wikipédia :

HTML (abreviação para a expressão inglesa HyperText Markup Language, que significa: "Linguagem de Marcação de Hipertexto") é uma linguagem de marcação utilizada na construção de páginas na Web. Documentos HTML podem ser interpretados por navegadores.

Tags HTML

Os marcadores (tags do inglês) são comandos de formatação da linguagem e aparecem entre chaves angulares (< tag >). Enquanto o HTML é responsável pela formatação, o estilo da página é definido usando CSS o qual não será abordado aqui.

Estrutura da página

A estrutura básica da página contém os elementos head and body

< !DOCTYPE html >
<html lang="pt-br" >
<head>
</head>
<body>
</body>
</html>

O DOCTYPE informa ao navegador o tipo do arquivo. A tag html define o inicio da página e a tag lang o idioma da página.

No head ficam os metadados, que são informações que não aparecem na sua página mas podem modificar a mesma. Por exemplo a tag title que é o título da página que aparece na aba do navegador. Aqui também se pode adicionar links para arquivos externos como um arquivo css.

No body ficam as tags e conteúdo que serão mostrados na sua página após a interpretada pelo navegador.

Elementos semânticos

Com a evolução do HTML (já está na versão 5), foram introduzidos elementos semânticos que trazem informações sobre o conteúdo dos elementos para o navegador e o desenvolvedor.

Um elemento como <div> não traz nenhuma informação sobre o conteúdo enquanto elementos como <form>, <table>, <section> claramente definem o conteúdo.

Por exemplo, dentro de body podemos dividir usando as tags header, que traz o cabeçalho da página, main, com o conteúdo principal e footer com o rodapé. Dentro de main podemos dividir em section, que compreende uma seção com conteúdos com mesmo tema ou sequenciais, e article que define cada conteúdo.

< body >
<header>
</header>
<main>
  <section>
   <article>
   </article>
   <article>
  </article>
  <article>
  </article>
 </section>
</main>
<footer>
</footer>
</body>

Exemplos de tags

Uma lista das tags pode ser encontrada no site w3schools.

  • <h1>, <h2> ... <h6>: Definem os títulos variando o tamanho da letra, <h1>, maior, para <h6> menor.
  • <p>: parágrafo
  • <br />: quebra de linha
  • <b>, <i>, <u> e <s> : negrito, itálico, sublinhado e riscado
  • <a>: hiperlink, local ou para página externa
  • <img>: imagem
  • <video>: conteúdo de video
  • <table>: tabela
  • <ol>: lista ordenada
  • <ul>: lista não ordenada
  • <li>: item da lista
